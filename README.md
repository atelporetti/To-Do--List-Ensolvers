# To-Do-List---Ensolvers

_Implementation Exercise_

Implement a simple web application that allows to create to-do items and folders to group them. 

### User Stories
- As a user, I want to be able to create, edit and delete to-do items
- As a user, I want to mark/unmark to-do items as completed
